# The Orange Library Web
Repo for [theorangelibrary.org](https://theorangelibrary.org) website. Stores all from HTML, JS, CSS, and images. Builds automatically and hosts on Github Pages.

Requires internet connection to load JQuery.

## Site map
[index.html](./index.html) contains the HTML content for the landing page ([theorangelibrary.org](https://theorangelibrary.org)).

[photos/index.html](./photos/index.html) contains the HTML content for the photo archive page ([theorangelibrary.org/photos](https://theorangelibrary.org/photos)). The images are served from the [photo-archive](./photo-archive) directory.
